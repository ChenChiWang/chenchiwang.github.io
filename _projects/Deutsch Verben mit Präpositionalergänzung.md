# 易混淆補充

### 1. 「照顧」與「擔心」 (Sorgen)

| 動詞結構 | 焦點 | A1-A2 程度例句 | 中文翻譯 |
| :--- | :--- | :--- | :--- |
| **sorgen für + A** | 負責/提供生活所需 | Ich sorge für das Abendessen. | 我負責準備晚餐。 |
| **sich sorgen um + A** | 心理上的憂慮 | Er sorgt sich um seine Zukunft. | 他擔心他的未來。 |
| **sich (D) Sorgen machen um + A** | 口語常用(擔心某事) | Mach dir keine Sorgen um mich! | 別為我擔心！ |

---

### 2. 「感謝」的對象與原因 (Danken / Bedanken)

| 動詞結構 | 重點 | A1-A2 程度例句 | 中文翻譯 |
| :--- | :--- | :--- | :--- |
| **sich bei + D bedanken** | 感謝的「對象」 | Ich bedanke mich bei Ihnen. | 我向您致謝。 |
| **danken für + A** | 感謝的「原因」 | Ich danke dir für die Hilfe. | 謝謝你的幫助。 |
| **sich bei + D für + A bedanken** | 對象 + 原因 | Sie bedankt sich bei ihm für das Geschenk. | 她為了禮物向他致謝。 |

---

### 3. 「高興」的時間差別 (Freuen)

| 動詞結構 | 時間點 | A1-A2 程度例句 | 中文翻譯 |
| :--- | :--- | :--- | :--- |
| **sich freuen auf + A** | 未來 (期待) | Wir freuen uns auf die Ferien. | 我們期待假期。 |
| **sich freuen über + A** | 現在/過去 (已發生) | Er freut sich sehr über den Sieg. | 他為這次勝利感到高興。 |

---

### 4. 「談論」的深淺程度 (Sprechen)

| 動詞結構 | 談論性質 | A1-A2 程度例句 | 中文翻譯 |
| :--- | :--- | :--- | :--- |
| **sprechen über + A** | 詳細討論某個主題 | Wir sprechen über das neue Projekt. | 我們討論這個新計畫。 |
| **sprechen von + D** | 簡略提到/說起 | Er hat kurz von seinem Bruder gesprochen. | 他稍微提到了他的兄弟。 |
